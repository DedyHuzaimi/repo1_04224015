# Dashboard Bencana Jawa Timur

Aplikasi berbasis PHP untuk menampilkan data bencana dari Open Data Jawa Timur.

## Fitur
- Menampilkan data bencana
- Filter berdasarkan tahun dan kategori
- Ranking daerah rawan bencana
- Grafik visual menggunakan Chart.js

## Sumber Data
API Open Data Jawa Timur

## Teknologi
- PHP
- Bootstrap
- Chart.js

## Cara Menjalankan
1. Jalankan server:
   php -S localhost:8000
2. Buka di browser:
   http://localhost:8000
